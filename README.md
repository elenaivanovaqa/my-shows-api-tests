## Автотесты на API проекта My Shows Rating
Проект My Shows Rating представляет из себя сервис для ведения личного рейтинга сериалов. Он состоит из микросервиса, nginx и базы Postgres, всё это связано с помощью docker compose и может быть одной командой развёрнуто локально.

### Описание проекта и задачи
Автоматизировать часть проверок регресса с помощью Pytest и Requests

Тесты написаны на языке `Python` с использованием фреймворка `Pytest`.
Приложение My Shows Rating разворачивается только в `Docker`, поэтому для одновременного развертывания приложения и запуска тестов реализован `Docker Compose`, где часть yml-файла для запуска приложения написана разработчиками приложения, а часть для запуска тестов (сервис `api-tests`) и сборка всего в единый файл написана мной.


#### Тест-кейсы, которые автоматизировали
* получение списка своих сериалов `GET /series`
* изменение данных конкретного сериала `PUT /api/v1/series/{id}`

#### Ожидаемый ответ:
* для обоих проверок response status code = 200
* тело ответа по запросу `GET /series` соответствует схеме JSON
* измененное значение данных сериала по запросу `PUT /api/v1/series/{id}` записалось в базу

### Детали реализации

* Автотесты написаны с применением Pytest
* Используется библиотека Requests
* Параметризированные тест и фикстура с использованием декоратора `@pytest.mark.parametrize`
* Подготовка тестовых данных через прямой запрос к БД при помощи библиотеки `psycopg`

### Запуск автотестов
Для запуска необходимо:
* предварительно скачать проект My Shows Rating (приложение находится в закрытом доступе)
* запустить Docker
* перейти в директорию с тестами
* собрать и запустить все сервисы по команде `docker-compose up --build`
